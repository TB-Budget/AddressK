Адрес субъекта - редактор

Общие сведения

Форма "Адрес субъекта" (проект "АдресК") заменяет одноименную форму стандартного комплекта, поставляемого разработчиком платформы "Турбо 9".

Форма отличается оптимизированным пользовательским интерфейсом, функционал формы обеспечивает возможность обращения к базе адресов (стандарт ФИОСМ), размещенную в сети интернет. По умолчанию адрес обращения http://fiosm.tb-b.ru. База адресов содержит сведения об адресных объектах, расположенных на территории Российской Федерации.

Содержимое полей ввода/вывода и порядок работы с формой

Форма содержит несколько информационных блоков (областей), визуально разделенных элементами-сепараторами.

Область "Сведения о субъекте"

Содержит код и наименование субъекта.

Блок "Актуальность"

Содержит поля для ввода начала / окончания времени, в течение которого сведения сохраяняли актуальность.

Поле Страна заполняется только для адреса субъекта за пределами РФ. Для адресов на территории РФ заполнение этого поля не обязательно.

Блок "Тип адреса"

Содержит набор чекбоксов, позволяющих отнести адрес к одному или нескольким типам. Набор доступных типов для юридических и физических лиц может быть различным.

Блок "Естественная запись"

Содержит единственное полее ввода/вывода, позволяющее сохранять адресную запись в естественном виде. Значение из этого поля может быть использовано для автоматизации оформления документов, не имеющих жестко регламентированных форм (договора, соглашения и т.п.).

Кнопка Обновить запускает процедуру автоматической генерации естественной формы записи из имеющихся структурных алементов адреса.

Блок "Дополнительная информация"

Поля ИМНС, Номер участка, Индекс, ОКАТО, ОКТМО, Телефон предназначены для прямого ввода. Некоторые из этих полей будут заполнены автоматически при заполнении блока Элементы адреса.

Поле Код KLADR - информационное, только для вывода. Значение формитруется автоматически при заполнении блока Элементы адреса.

Блок "Элементы адреса"

Блок представляет собой таблицу, содержащую элементы адреса согласно стандартам КЛАДР / ФИОСМ.

Поля в строках:

  • Регион
  • Округ
  • Район
  • Город (волость, сельский округ...)
  • Внутригородская территория
  • Населенный пункт
  • Улица
  • Дополнительная территория
  • Дополнительные объекты
имеют кнопки обзора. Нажатие на кнопку обзора запускает процедуру поиска элемента в базе данных. Количество элементов может быть значительным. В отдельных случаях это количество может быть столь вилико, что будет приводить к переполнению буфера обмена. Для избежания этого вводите в поля ввода первые 2-4 символа наименования и используйте клавишу Enter или кнопку обзора. Время поиска в этом случае существенно сократится, объем извлекаемой информации не будет приводить к переполнению буфера обмена.

6-й уровень иерархии в базе адресов России предполагает выбор типа объекта. Обычно это "ДОМ". Номер дома, владения и т.п. выбирается из выпадающего списка или вводится вручную.

Элемент "Номер квартиры / офиса" предполагает только ручной ввод.
