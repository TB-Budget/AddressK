Тип адреса

Общие сведения

Справочник "Тип адреса" - это простая картотека, не имеющая бланка-редактора.

Стандартных записей, содержащихся в этом справочнике, как правило, достаточно для описания адреса субъекта. При этом добавление пользовательских типов адресов не запрещено.

Создание пользовательсткого типа адреса

Для добавления пользовательского типа адреса:

  • Создайте новую запись
  • Введите код. Код - произвольная последовательность символов, не содержащая пробелов и специальных символов.
  • Введите наименование
  • Взведите флаг "Для физлиц" и (или) "Для контрагентов". Указанные флаги регулируют состав возможных типов адресов в карточке адреса контрагента и карточке адреса физлица, не являющегося контрагентом. При сброшенных флагах изменение этого типа адреса в карточке адреса будет недоступно.
  • Укажите порядок вывода типа адреса в списке типов адресов. Порядок вывода - это целочисленное значение. При значении 0 этот тип адреса будет выведен первым в списке типов адресов.
